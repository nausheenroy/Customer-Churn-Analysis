# Customer Churn Analysis Dashboard (Tableau)

## Overview
This project analyzes customer churn behavior and its impact on revenue using Tableau. 
The dashboard provides insights into churn rate, revenue from churned customers, and churn patterns across contract types, payment methods, tenure, and internet services.

## Key Metrics
- Total Customers
- Churn Rate
- Revenue from Churned Customers

## Key Insights
- Month-to-month contracts show the highest churn impact.
- Customers with higher monthly charges are more likely to churn.
- Churned customers contribute significantly to revenue loss.

## Tools Used
- Tableau Desktop
- Data Visualization
- Business Analytics
- Jupyter Notebook(EDA and Data Cleaning)
